Microfi Free Flow

Bitcoin Cash CashTokens paper (gift) wallet

This is a client-side wallet for demonstration purposes.
Microfi Free Flow is not Virtual Asset Service Provider.
Download the source code from Github and run it locally.

Send min 1600 satoshi (0.000016 BCH) and only one category of CashTokens to a paper wallet and share the private key of the paper wallet with someone (e.g. as a gift).
The private key of the paper wallet can be swept in the Microfi wallet (https://microfi.eu/wallet/).
